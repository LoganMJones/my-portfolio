# Assets folder: put your media files here

Upload via GitHub: **Add file → Upload files** into this folder.

Each layout has a **blue fill checklist** in `index.html` with exact filenames and minimum image sizes. Open your layout and search `fill-checklist`.

## Quick reference

| File type | Example names | Tips |
|-----------|---------------|------|
| Headshot | `photo.jpg` | 400×400 px min, square, JPG |
| Hero / banner | `hero.jpg` | 1200–1600 px wide, JPG |
| Project images | `project-1.jpg`, `project-featured.jpg` | 800×450 px min for wide slots |
| Gallery | `gallery-1.jpg` … `gallery-6.jpg` | 300×300 px min per image |
| CV / papers | `cv.pdf`, `paper.pdf`, `slides.pdf` | Under 2 MB each |
| Video | (none) | Use a YouTube or Vimeo embed; no file upload |

**Compress images** at [squoosh.app](https://squoosh.app) before uploading (free, no account).

**Privacy:** only upload files you're allowed to share publicly. See the [privacy guide](https://loganmjones.github.io/PortfolioTraining/guide.html#privacy).
